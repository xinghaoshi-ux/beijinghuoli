# 状态与异常路径

## 1. 公共端状态

### 1.1 页面加载

触发场景：

- 首次进入公共端。
- 切换 tab。
- 输入搜索关键词。

页面表现：

- 显示加载状态。
- 保持页面结构稳定。

### 1.2 数据为空

空状态：

| 模块 | 文案 |
|---|---|
| 比赛结果 | 暂无比赛结果 |
| 球队排名 | 暂无球队排名 |
| 个人排名 | 暂无个人排名 |

### 1.3 搜索无结果

触发场景：

- 用户输入关键词后没有匹配数据。

页面表现：

- 显示“未找到匹配结果”。
- 提供清空搜索入口。

### 1.4 接口异常

触发场景：

- 后端接口不可用。
- 网络异常。
- 接口返回错误。

页面表现：

- 显示简短错误提示。
- 提供重新加载入口。
- 不展示假数据。

## 2. 后台状态

### 2.1 表单校验失败

触发场景：

- 必填项为空。
- 积分不是合法数字。
- 未选择球员或球队。

页面表现：

- 在字段附近展示错误提示。
- 不提交请求。

### 2.2 保存中

触发场景：

- 新建或编辑球员。
- 新建或编辑球队。
- 保存成绩。

页面表现：

- 禁用重复提交。
- 按钮显示保存中状态。

### 2.3 上传中

触发场景：

- 上传 Excel 文件。

页面表现：

- 显示上传中。
- 禁用重复上传。

### 2.4 解析失败

触发场景：

- Excel 格式不正确。
- 字段缺失。
- 数据无法匹配。

页面表现：

- 显示失败原因。
- 允许重新上传。

### 2.5 解析预览

触发场景：

- Excel 解析成功。

页面表现：

- 展示预览表。
- 标出正常行和错误行。
- 错误行不能确认导入。

### 2.6 确认导入失败

触发场景：

- 后端写入失败。
- 预览数据状态无效。

页面表现：

- 显示错误提示。
- 保留预览数据，允许用户重试或取消。

## 3. 成绩修正异常

### 3.1 编辑成绩失败

可能原因：

- 成绩不存在。
- 球员或球队不存在。
- 积分不合法。

处理：

- 展示错误提示。
- 不修改本地展示数据。

### 3.2 删除成绩

是否允许删除仍待确认。

如果允许删除：

- 删除前二次确认。
- 删除后同步删除对应积分记录。
- 删除后刷新结果和排名。

## 4. 禁止行为

- 公共端不得在接口异常时展示伪造排名。
- 后台不得绕过接口直接修改前端状态冒充保存成功。
- Excel 错误行不得直接导入。
- 未确认的删除能力不得提前开放入口。
