# 项目边界说明

## 1. 项目名称

北京市活力网球交流系列赛赛果查看平台

## 2. 项目定位

本项目是一个独立的单场赛事赛果积分查看平台，用于支持交流赛期间和赛后的结果查询、球队积分查询、个人积分查询，以及后台基础数据录入。

本项目不直接改造原有年度赛事积分系统。后续如需求稳定，可再评估数据模型、API 和页面是否整合回原系统。

## 3. 目标用户

公共端用户：

- 参赛球员
- 参赛球队成员
- 赛事组织者
- 关注比赛结果的相关人员

后台用户：

- 赛事管理员
- 成绩录入人员

## 4. 本版范围

公共端：

- 比赛结果查询
- 球队排名与积分
- 个人排名与积分
- 支持 Web 端和手机端访问

后台：

- 球队录入
- 球员录入
- 成绩录入

## 5. 本版不做

- 多赛季
- 多赛事切换
- 球员个人主页
- 球队主页
- 数据导出
- 操作日志或数据变更记录
- 团体积分分摊到个人
- 复杂筛选和二级排序
- 与原年度积分平台直接集成

## 6. 核心业务约束

- 本项目只服务一个交流赛。
- 排名分为个人排名和球队排名。
- 个人积分和球队积分相互独立。
- 团体或球队积分暂不分摊到个人。
- 同分并列，下一名跳号。
- 后台成绩录入需要支持手工录入和 Excel 上传。

## 7. 后续预留

后续可扩展：

- 球员主页
- 球队主页
- 数据导出
- 多赛事或多届赛事
- 与原年度积分系统整合
- 更复杂的积分规则
