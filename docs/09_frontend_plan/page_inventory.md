# 页面清单

## 1. 公共端页面

### U-01 公共查询首页

路径建议：`/`

用途：

- 展示赛事名称。
- 提供统一搜索。
- 提供比赛结果、球队排名、个人排名三个 tab。

包含模块：

- 赛事标题区
- 搜索框
- tab 导航
- 比赛结果列表
- 球队排名列表
- 个人排名列表

数据来源：

- 公共比赛结果接口
- 公共球队排名接口
- 公共个人排名接口

### U-02 公共错误页，可选

路径建议：`/error`

用途：

- 当前端接口不可用或发生异常时展示。

本页可在 MVP 中通过页面内错误态替代，不一定单独实现。

## 2. 后台页面

### A-01 登录页

路径建议：`/admin/login`

用途：

- 后台登录。

说明：

- 是否启用后台登录待确认。
- 如不启用，可不实现此页面。

### A-02 后台概览

路径建议：`/admin`

用途：

- 展示当前数据概况。

展示内容建议：

- 球员数量
- 球队数量
- 成绩数量
- 个人排名前三
- 球队排名前三

### A-03 球员管理

路径建议：`/admin/players`

用途：

- 查看、新建、编辑球员。

核心组件：

- 搜索框
- 球员表格
- 新建/编辑表单

### A-04 球队管理

路径建议：`/admin/teams`

用途：

- 查看、新建、编辑球队。
- 维护球队成员。

核心组件：

- 搜索框
- 球队表格
- 新建/编辑表单
- 成员管理抽屉或弹窗

### A-05 成绩录入

路径建议：`/admin/results/new`

用途：

- 手工录入成绩。
- Excel 上传成绩。

核心组件：

- 录入方式切换
- 手工录入表单
- Excel 上传控件
- Excel 解析预览
- 确认导入按钮

### A-06 成绩管理

路径建议：`/admin/results`

用途：

- 查看、搜索、编辑已录入成绩。

核心组件：

- 搜索框
- 成绩表格
- 编辑表单

### A-07 排名预览

路径建议：`/admin/rankings`

用途：

- 后台查看当前个人排名和球队排名。

核心组件：

- 个人排名 tab
- 球队排名 tab
- 排名表格

## 4. 非本版页面

本版不做：

- 球员主页
- 球队主页
- 多赛事页面
- 多赛季页面
- 数据导出页面
- 操作日志页面
