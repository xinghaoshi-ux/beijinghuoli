## 5. 开发顺序建议

| 批次 | 模块 | 原因 |
|---|---|---|
| 第 1 批 | 数据模型与 API 契约 | 后续实现依据 |
| 第 2 批 | 后端球员、球队、成员管理 | 成绩录入前置 |
| 第 3 批 | 后端成绩录入与排名查询 | 主链路核心 |
| 第 4 批 | 后台管理端 | 支撑真实录入 |
| 第 5 批 | 公共端 Web 和手机端 | 支撑用户查询 |
| 第 6 批 | 联调、测试、部署 | 形成可用版本 |

## 6. 待确认项

| 编号 | 待确认项 | 影响 |
|---|---|---|
| Q-01 | 是否启用后台登录 | 影响后端鉴权和后台入口 |
| Q-02 | 成绩是否记录比分 | 影响结果模型和录入表单 |
| Q-03 | Excel 模板字段 | 影响上传解析 |
| Q-04 | 数据库是否使用 SQLite | 影响初始化和部署 |
| Q-05 | 是否允许删除成绩 | 影响后台修正流程 |
